# Security Policy

## Supported Versions

We only support the newest versions. (Probably.)

| Version | Supported           |
| ------- | ------------------  |
| 5.1.x   | ✅                 |
| 5.0.x   | ❌                 |
| 4.0.x   | ❌                 |
| < 4.0   | ❌                 |

## Reporting a Vulnerability

If you find a vulnerability, it will be your job to help us ensure that it is fixed. SUPPORT SERVER: https://discord.gg/nhht5PE4zS
You can expect an update on your vulnerability as soon as a member of staff can deal with and verify it. If the vulnerability is found,
you will need to provide them with the code fixes to change it. (Note that the code is checked for anything malicious before being used.)
If the vulnerability report is declined, the ticket channel will be closed and you will be notified.
